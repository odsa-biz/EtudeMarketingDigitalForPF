🎯 Créneau disponible identifié

Le segment "artisan local qualifié, réactif sous 48h, transparent sur les prix" est sous-occupé à Paris 14e. Martin est trop gros pour être réactif. Dépann'Express est trop cher. Samir est trop chargé. Ce créneau existe — il faut juste le rendre visible.

Plan d'Action
30 · 60 · 90 jours.
Chaque action dans le bon ordre.

Pas de théorie. Des actions concrètes, classées par impact, applicables dès la semaine suivante.

J1 — J30
Rendre visible ce qui existe déjà
Les fondations. Impact immédiat, zéro budget pub.
  • Optimiser la fiche Google Business Profile (photos, services, description clé)
  • Collecter 20 premiers avis via SMS aux meilleurs clients existants
  • Mettre en place la relance devis automatique J+2 via WhatsApp Business
  • Créer un template devis pro avec signature numérique
J30 — J60
Automatiser les tâches répétitives
Gagner du temps. Chaque intervention génère un avis.
  • Automatiser la demande d'avis Google post-intervention (workflow Make)
  • Page de présentation simple avec portfolio photos avant/après
  • Inscription Houzz et MeilleurArtisan avec les mêmes visuels GBP
  • Analyser les devis perdus du mois — identifier le pattern de perte
J60 — J90
Ancrer, mesurer, dupliquer
On mesure, on ajuste, on accélère ce qui marche.
  • Vérifier l'apparition dans ChatGPT sur "plombier Paris 14"
  • Tableau de bord simple : appels, devis, taux de transformation
  • Identifier les 3 types de chantiers les plus rentables à cibler
  • Ajustement du plan pour le trimestre suivant
